Testing the Model¶

Using income as level, actual percentage change ($\textbf{%c}$) per MSA for each variable and the estimated coefficients $\alpha$ and $\beta$ are used to derive model predictions ($\textbf{Model Pred}$) and the difference between the prediction and actual change in area is caluculated ($\textbf{Diff}$).

In [78]:
ule = df[['Area','GMP','Population']].copy()
ule['%c_Area'] = (ule.groupby('NAME')['Area'].apply(pd.Series.pct_change))
ule['%c_GMP'] = (ule.groupby('NAME')['GMP'].apply(pd.Series.pct_change))
ule['%c_Pop'] = (ule.groupby('NAME')['Population'].apply(pd.Series.pct_change))
ule = ule[['Area','%c_Area','GMP','%c_GMP','Population','%c_Pop']]
alpha = .5651
beta = .9624
ule['Model Pred'] = (alpha-beta) * ule['%c_Pop']+ule['%c_GMP']
ule['sd_Residuals'] =  (ule['Model Pred'] - ule['%c_Area']) ** 2
fin = ule[['Area','%c_Area','%c_Pop','%c_GMP','Model Pred','sd_Residuals']]
